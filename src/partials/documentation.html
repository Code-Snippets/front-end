Code Snippets

Snippets

The snippets are placed in the /snippets folder and on build their name, location and description are placed in /snippets.json

Search

Code snippets uses a fuzzy search algorithm to match the name and description of the snippet.

Make a snippet

In order to make a snippet you need to go to Tools > Make a snippet, fill the form, copy the code a the end of the page, make a new file in the snippets folder and make a pull request.
If you store a local copy of this app you will need to run gulp build to add the new snippets in the snippets.json file.

Form options

  • Name - The name of your snippet (used for fuzzy search)
  • Description - The description of your snippet (used for fuzzy search)
  • Panels - A panel is the box where the code will be displayed on the snippet page
    • Code Panel - Its input will be highlighted
    • ProTip Panel - Its input will be compiled as markdown.
  • Language - Specifies the language to be highlighted for the code panel.
  • Url(optional) - If present this url will be used to replace the content of the panel with the one fetched from the url. If the url fails the content specified in the panel will be used as a fallback.
  • Code - The content of the panel.
  • Has cursors(optional) - If this is true it will parse the code for cursors. They will allow to have a dynamic snippet based on the user input.

You can add as many panels as you want.

Keyboard shortcuts

The available shortcuts are:

  • s - focus on the search input
  • Tab - focus on the next result(in the search page)
  • Up, Down, j, k - Next/Previous result (in the search page)
